# APIELIB - API Thư viện số

ASP.NET Core Web API cho hệ thống quản lý thư viện số (Electronic Library), sử dụng Entity Framework Core.

## Chức năng chính
- **Search API**: Tìm kiếm tài liệu với nhiều tiêu chí
- **GetBook API**: Lấy thông tin chi tiết tài liệu

## Công nghệ
- .NET 8.0
- Entity Framework Core
- SQL Server
